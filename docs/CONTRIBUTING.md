# Contributing Guide

## Linting and Formatting

Run ESLint with Prettier to check code style:

```bash
npm run lint
```

To automatically fix common issues, use:

```bash
npm run lint -- --fix
```

Configure your editor to format on save using the repo's Prettier and ESLint settings.
